NON-CONFORMITY REPORT: NCR-2023-0101
================================================================================

PRODUCT: NEXUS CK MB (Reference 30421)
LOT NUMBER: 1009900941
DATE OPENED: 2023-07-13
STATUS: Investigation

TITLE: primary signal out of specification - second consecutive batch

DESCRIPTION:
During calibration verification for lot 1009900941, primary signal reading of
578.3 SIU exceeded upper specification limit of 550 SIU. This is the second
consecutive batch showing Primary OOS results (see NCR-2023-0100 for first
occurrence on lot 1009900940).

RELATED NCR: NCR-2023-0100 (Lot 1009900940, primary signal = 572.5 / 575.8 SIU)

SYMPTOM:
primary signal too strong, exceeding upper specification limit. Error code B18
(Invalid calibration) generated. Pattern consistent with NCR-2023-0100 -
systematic upward shift in primary signal.

TEST RESULTS:
  Initial Test (2023-07-13):
    Primary Signal: 578.3 SIU (Spec: 450-550 SIU) - FAIL
    Secondary Signal: 100.9 SIU (Spec: 80-120 SIU) - PASS
    Error Code: B18 (Invalid calibration - Primary out of range)
    Instrument: NEXUS-002 (EQ-CAL-002)
    Analyst: QC-003 (A. Lawson)

NOTE: Test performed on different instrument (NEXUS-002 vs NEXUS-001 for
NCR-2023-0100) to rule out instrument-specific bias. Same OOS pattern
observed, further supporting material-related root cause.

IMPACT ANALYSIS:
Second consecutive batch affected. Combined quarantined quantity: approximately
5,000 kits. Production of NEXUS CK MB remains on hold. Pattern confirmation
increases urgency of investigation.

PROCESS STEP: Calibration

IMMEDIATE ACTION:
- Lot 1009900941 quarantined (2023-07-13)
- Production hold on NEXUS CK MB maintained
- All remaining batches using CART lot CART-2023-06-002 flagged for enhanced
  monitoring
- Escalation to Quality Director

PATTERN ANALYSIS:
Two consecutive batches (1009900940, 1009900941) both show primary signal
approximately 25-30 SIU above upper specification limit. secondary signal remains
normal in both batches. Both batches use CART coating lot CART-2023-06-002,
which was qualified under CR-2023-0087 and introduced starting with batch
1009900940. All prior batches using CART lot CART-2023-05-001 showed normal S1
values (range: 495-510 SIU).

ROOT CAUSE: Under investigation - material variation in CART coating lot
CART-2023-06-002 (elevated reactive surface capacity)

CAPA REQUIRED: Yes (linked to NCR-2023-0100 CAPA)
CAPA REFERENCE: Pending investigation completion
