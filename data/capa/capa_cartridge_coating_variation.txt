1. PROBLEM DESCRIPTION
================================================================================

Multiple consecutive NEXUS CK MB production lots (starting with 1009900940)
exhibited primary calibration signal exceeding upper specification limit of 550 SIU,
resulting in calibration failure (error code B18). Investigation identified
root cause as elevated capture reagent density in CART coating lot CART-2023-06-002
(2.4 mg/mL, at upper end of 2.0-2.5 specification range) from supplier
Surface Coating Solutions.

IMPACT:
  - 5 lots quarantined (1009900940 through 1009900944)
  - Production hold for NEXUS CK MB: 3 weeks
  - Estimated financial impact: EUR 225,000
  - No customer-reaching product (all lots caught during QC)

ROOT CAUSE:
CART coating batch CART-2023-06-002 has elevated reactive surface capacity due
to higher capture reagent density (2.4 mg/mL vs typical 2.1-2.2 mg/mL). The higher
density results in increased target analyte capture on sensor cartridges, producing elevated
primary optical signal that exceeds the 550 SIU upper specification limit.

ROOT CAUSE CATEGORY: Matieres (Materials) - Supplier lot variation

================================================================================
3. PREVENTIVE ACTIONS
================================================================================

PA-1: Tighten Incoming Inspection Criteria
  Action: Revise incoming inspection specification for NEXUS CK MB sensor cartridges.
  Reduce capture reagent density acceptance range from 2.0-2.5 mg/mL to 2.0-2.3
  mg/mL. Values between 2.3-2.5 mg/mL require QC Supervisor approval and
  enhanced initial batch monitoring.
  Owner: A. Lawson (QC Supervisor)
  Due Date: 2023-09-30
  STATUS: COMPLETED (2023-09-15)
  Evidence: Updated SPEC-CART-CKMB Rev 5

PA-2: Enhanced First-Batch Monitoring
  Action: Implement mandatory enhanced monitoring protocol for first 3 batches
  after any CART lot change. Enhanced monitoring includes: duplicate calibration
  testing, comparison to previous lot Primary signal values, and QC Supervisor review
  before batch release.
  Owner: M. Payne (QC Analyst)
  Due Date: 2023-09-30
  STATUS: COMPLETED (2023-09-20)
  Evidence: Updated SOP-CAL-001 Rev 4 (new Section 8.5)

PA-3: Supplier Specification Alignment
  Action: Require Surface Coating Solutions to tighten internal capture reagent density
  specification to 2.0-2.3 mg/mL (aligned with updated MedTech Diagnostics spec).
  Include as condition for continued supplier qualification.
  Owner: N. Carter (Supplier Quality Engineer)
  Due Date: 2023-12-31
  STATUS: IN PROGRESS
  Update: Surface Coating Solutions agreed in principle during September 2023 audit.
  Formal specification change expected by November 2023.

PA-4: QC Trend Analysis Enhancement
  Action: Add correlation analysis between CART lot identifier and primary signal
  values in QC trend analysis system. Automated alert if Primary signal values for a new
  CART lot deviate by more than 2 sigma from historical mean.
  Owner: M. Payne (QC Analyst)
  Due Date: 2023-10-31
  STATUS: COMPLETED (2023-10-25)
  Evidence: Trend analysis system updated, validation report QC-VAL-2023-015
